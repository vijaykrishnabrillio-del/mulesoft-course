☕ Step 1: Install Java JDK

Why Java?

Anypoint Studio is built on Eclipse and requires Java Development Kit (JDK) to run.

Check Existing Installation:

# Open terminal/command prompt and run:
java -version
javac -version

Expected Output:

java version "1.8.0_xxx" or "11.0.x"
Java(TM) SE Runtime Environment
javac 1.8.0_xxx or 11.0.x

If Java is NOT Installed:

Windows:

  1. Download AdoptOpenJDK 8 or 11 from adoptopenjdk.net
  2. Choose:

    3. - Version: OpenJDK 8 (LTS) or 11 (LTS)

    - JVM: HotSpot

    1. Run installer (choose "Add to PATH" option)
    2. Verify installation with java -version

      3. macOS:

      # Using Homebrew
brew install openjdk@11
# Add to PATH
echo 'export PATH="/usr/local/opt/openjdk@11/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

      Linux (Ubuntu/Debian):

      sudo apt update
sudo apt install openjdk-11-jdk
java -version

      Set JAVA_HOME Environment Variable:

      Windows:

      1. Search "Environment Variables" in Start menu
2. Click "Environment Variables" button
3. Under "System Variables", click "New"
   - Variable name: JAVA_HOME
   - Variable value: C:\Program Files\Java\jdk1.8.0_xxx
4. Click OK

      macOS/Linux:

      # Add to ~/.bashrc or ~/.zshrc
export JAVA_HOME=$(/usr/libexec/java_home)  # macOS
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64  # Linux

      📦 Step 3: Install Anypoint Studio

      Windows Installation:

      1. Extract the downloaded .zip file
      2. Move the AnypointStudio folder to C:\ (recommended)
      3. Navigate to C:\AnypointStudio
      4. Run AnypointStudio.exe

        5. macOS Installation:

        1. Open the downloaded .dmg file
        2. Drag Anypoint Studio to Applications
        3. Right-click → Open (first time only, to bypass Gatekeeper)
        4. Grant permissions if prompted

          5. Linux Installation:

          # Extract archive
tar -xzf AnypointStudio-*.tar.gz

# Move to /opt (optional)
sudo mv AnypointStudio /opt/

# Run Studio
cd /opt/AnypointStudio
./AnypointStudio

                    ⚙️ Step 5: Configure Studio Preferences

                    Recommended Settings:

                    1. Font Size (for better readability):

                    Window → Preferences → General → Appearance → Colors and Fonts
→ Basic → Text Font → Edit → Size: 12-14

                    2. Auto-Save:

                    Window → Preferences → General → Editors → Autosave
☑ Enable autosave every 30 seconds

                    3. Maven Settings:

                    Anypoint Studio → Preferences → Anypoint Studio → Maven
☑ Use embedded Maven installation

                    4. Runtime Manager:

                    Anypoint Studio → Preferences → Anypoint Studio → Authentication
→ Sign in with Anypoint Platform credentials

                    🚨 Common Installation Issues & Troubleshooting

                    Issue 1: "Java version not compatible"

                    Solution:

                    • Ensure JDK 8 or 11 is installed (not JRE)
                    • Set JAVA_HOME correctly
                    • Restart Studio after Java installation

                    Issue 2: Studio won't start

                    Solution:

                    1. Check AnypointStudio.ini file
2. Add/verify:
   -vm
   C:/Program Files/Java/jdk1.8.0_xxx/bin/javaw.exe

                    Issue 3: Workspace errors

                    Solution:

                    • Close Studio
                    • Delete .metadata folder in workspace
                    • Restart Studio (will recreate workspace)

                    Issue 4: Slow performance

                    Solution:

                    • Increase memory in AnypointStudio.ini:
                      • -Xms1024m
-Xmx4096m
