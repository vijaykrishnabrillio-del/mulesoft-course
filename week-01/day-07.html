Section C: Code Understanding (10 questions)

31. What does this DataWeave expression return?

  • A) The customer name or "Unknown" if null
  • B) Always "Unknown"
  • C) An error if customer is missing
  • D) The entire payload

32. What is the value of vars.result after this sequence?

<set-variable variableName="num1" value="10"/>
<set-variable variableName="num2" value="5"/>
<set-variable variableName="result" value="#[vars.num1 + vars.num2]"/>
  • A) "105"
  • B) 15
  • C) "10 + 5"
  • D) Error

33. This flow will respond with:

<flow name="test">
    <http:listener path="/test"/>
    <set-payload value="Hello"/>
    <logger message="#[payload]"/>
    <set-payload value="World"/>
</flow>
  • A) "Hello"
  • B) "World"
  • C) "HelloWorld"
  • D) Empty

34. How many loggers will execute?

<flow name="main">
    <http:listener path="/test"/>
    <logger message="1"/>
    <async>

</flow>
  • A) 2 (1 and 3)
  • B) 3 (all of them)
  • C) 1 (only first)
  • D) Depends on timing

  • B) "greeting Alice"
