Sample Questions

Question 1: DataWeave

Which DataWeave operator is used for string concatenation?

  • A) +
  • B) ++
  • C) &
  • D) concat
Show Answer

Answer: B) ++

Example: "Hello" ++ " World" results in "Hello World"

Question 2: HTTP Listener

What is the default port for HTTP Listener in Anypoint Studio?

  • A) 8080
  • B) 8081
  • C) 9090
  • D) 80
Show Answer

Answer: B) 8081

Question 3: API-Led Connectivity

In API-led connectivity, which layer handles business process orchestration?

  • A) Experience API
  • B) Process API
  • C) System API
  • D) Integration API
Show Answer

Answer: B) Process API

Process APIs orchestrate and compose data from multiple System APIs.

Question 4: Error Handling

What is the difference between On-Error-Continue and On-Error-Propagate?

  • A) No difference
  • B) Continue proceeds normally, Propagate re-throws error
  • C) Continue is for warnings, Propagate for errors
  • D) Continue logs error, Propagate ignores it
Show Answer

Answer: B) Continue proceeds normally, Propagate re-throws error

On-Error-Continue handles the error and continues flow execution. On-Error-Propagate handles the error but propagates it to the calling flow.

Question 5: CloudHub

What must HTTP Listener port be configured as for CloudHub deployment?

  • A) 8081
  • B) 80
  • C) ${http.port}
  • D) Any port
Show Answer

Answer: C) ${http.port}

CloudHub dynamically assigns a port, so you must use the property placeholder.
