Troubleshooting

Common Issues

1. "Port already in use"

Solution: Change the port in HTTP Listener config or stop the process using that port.

# Windows
netstat -ano | findstr :8081
taskkill /PID [process_id] /F

# Mac/Linux
lsof -ti:8081 | xargs kill -9

2. "Java version not compatible"

Solution: Ensure JDK 8 or 11 is installed and JAVA_HOME is set correctly.

java -version
echo %JAVA_HOME%  # Windows
echo $JAVA_HOME   # Mac/Linux

3. "Cannot coerce" DataWeave error

Solution: Use type coercion explicitly.

// Wrong
payload.age + 5

// Correct
(payload.age as Number) + 5
